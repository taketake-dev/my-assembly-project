# CODING_STYLE

## アセンブリ

### インデント

- ラベルは先頭に書くこと。
- インデントはスペース 4 つで行うこと。
- 命令とオペランドの間はスペース 5 つでインデントすること。
- コメントはラベル内の最も右の列からスペース 8 つ開けて書くこと。

```asm
ft_add:
    mov    rax, rdi       ; rax に第1引数をセット
    add    rax, rsi       ; rax に結果を格納
```

## ヘッダーファイル

### 読み込み

以下を使うこと

```c
#ifndef LIBASM_H
#define LIBASM_H
#endif /* LIBASM_H */
```

モダンな `#pragma once` とも書けるが、アセンブリを使うため古いコンパイラにも対応した書き方をする。
